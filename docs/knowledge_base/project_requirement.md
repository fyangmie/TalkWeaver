# Project Requirements Knowledge Base

TalkWeaver is a research-inspired final project centered on:

1. speaker diarization;
2. overlapping speech and cross-speech;
3. LLM and ASR synergy;
4. paper-driven component adaptation;
5. experiments and error analysis.

Local audio preprocessing is a supporting module. RAG is an auxiliary
domain-term correction mechanism and a secondary source for meeting
understanding.

The project must run in deterministic mock mode without a GPU or API tokens,
must preserve correction audit trails, and must never present mock metrics as
real experimental findings.
