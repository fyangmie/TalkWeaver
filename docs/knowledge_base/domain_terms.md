# TalkWeaver Domain Terms

This glossary supports ASR term recovery. Retrieved terms are candidates, not
permission to add unsupported facts.

| Term | Meaning | Common ASR confusion |
| --- | --- | --- |
| pyannote.audio | Toolkit for speaker diarization workflows | piano note |
| speaker diarization | Determining who spoke when | diary station |
| overlapping speech | Simultaneous speech from multiple speakers | overlap speech |
| cross-speech | Interruptions or simultaneous speaker activity | cross speech |
| ASR | Automatic speech recognition | letters expanded incorrectly |
| WER | Word error rate | where |
| DER | Diarization error rate | the ear |
| WDER | Word-level diarization-related error metric | double-u der |
| RAG | Retrieval-augmented generation | rack |
| faster-whisper | Efficient Whisper inference implementation | faster whisper |
| VAD | Voice activity detection | V A D |
| LLM correction | Constrained transcript post-processing | language correction |
| temporal anchor | Timestamp grounding for transcript content | time anchor |
| speaker attribution | Assigning content to a speaker | speaker contribution |

## Correction Pairs

```text
piano note -> pyannote
diary station -> diarization
where -> WER
the ear -> DER
rack -> RAG
```
